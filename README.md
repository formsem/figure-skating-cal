# ISU Figure Skating Score Calculator

## Overview

This project is a web-based Figure Skating Score Calculator designed to calculate the Base Value (BV) and Grade of Execution (GOE) for figure skating elements based on the ISU Judging System.

The calculator is built as a single-page HTML application and focuses on providing a fast and mobile-friendly scoring experience for:

* Single skating
* Jump combinations
* GOE calculation
* Technical element score tracking
* ISU-style element input workflow

The application is optimized for both desktop and mobile browsers.

---

## Features

### ISU Judging System Support

* Base Value calculation for figure skating elements
* GOE adjustment support
* Positive and negative GOE buttons
* Combination jump support
* Real-time score updates

### User Interface

* Mobile-friendly responsive design
* Modern glassmorphism inspired UI
* Fast element selection
* Color-coded GOE controls
* Sticky navigation header

### Technical Features

* Pure HTML, CSS, and JavaScript
* No backend required
* Lightweight and easy to deploy
* Works directly in the browser

## Getting Started

### Run Locally

Simply open the `index.html` file in your browser.

Or use a local server.

Example with VSCode Live Server:

1. Install the Live Server extension
2. Right-click `index.html`
3. Select:

```text
Open with Live Server
```

---

## GitHub Pages Deployment

This project can easily be hosted using GitHub Pages.

### Steps

1. Push the repository to GitHub
2. Open repository settings
3. Navigate to:

```text
Settings → Pages
```

4. Select:

```text
Branch: main
Folder: /root
```

5. Save changes

GitHub Pages will publish the calculator automatically.
